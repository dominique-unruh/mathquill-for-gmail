Hotkey:
 Syntax: [ctrl-][alt-][shift-]x where x is a single letter (a-z). (Only letters are supported in hotkeys.)
LaTeX rendering service:
 URL of service for rendering math. The URL must contain @@@ which will be replaced by the LaTeX code to be rendered.
Macros:
 Macros for abbreviating common subformulas. One per line. See the documentation.
Style:
 CSS styling to apply to the images. (Width, height, and vertical-align are set automatically.)
